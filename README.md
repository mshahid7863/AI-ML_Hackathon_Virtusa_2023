# AI-ML_Hackathon_Virtusa_2023

Automate Test Automation with (AI/ML)" is a project that leverages Flask and Streamlit frameworks for application development. By integrating OpenAI API, the project offers an automatic code generation feature. This innovation eliminates the need for manual scripting or recording, thereby significantly reducing the learning curve and effort required for beginners in automation scripting. The project aims to cut down script development time by 50% for novices who lack knowledge of test script creation.
